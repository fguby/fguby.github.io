树莓派实现远程控制空调

发表于2022-01-16|更新于2022-01-16|树莓派
|字数总计:667|阅读时长:2分钟|阅读量:

家里老式空调的遥控器最近找不到了，正好家里还有一个闲置的树莓派，之前在北京的时候用它远程控制过空调，正好拿来废物利用了。

该篇博文搬运自我之前的CSDN博客，如有错误，欢迎指正。

我买的树莓派是3代B +（淘宝京东很多），以下操作也全都是基于这个版本的树莓派。

关于系统

安装系统这一步，网上能找到很多教程，用TF卡随便装一个你想要装的系统，然后插好电源，连接显示器开机就行。（我就不写那么多废话了）

我使用的系统是 RASPBIAN（2018年4月的版本）

如果你想查看树莓派的ip地址，命令行输入命令hostname -I就行。(无屏幕小伙伴自行google)

首先，开启SSH以及VNC。

  1. 命令行输入sudo raspi-config。

  2. 找到Interfacing Options一行，回车。

  3. 你会看到SSH和VNC，然后都修改为yes就行。

  4. 选择OK，Finish。

这个时候你就可以试试用ssh或者vnc远程连接你的树莓派了。

无公网ip实现远程连接

在第一步的基础上，你开启了SSH和VNC，但是很快就会发现在公司还是无法远程连接上你的树莓派，因为你的树莓派分到的只是一个内网ip。

关于这一点，我采用的解决方法是Sunny-Ngrok

官网地址: https://www.ngrok.cc
教程地址: https://www.sunnyos.com/article-show-67.html

简单，好用，最主要是免费。

红外接收器和发射器

我使用的是一款IR红外控制扩展板，淘宝一搜就有。

配置完扩展板之后，首先需要安装Lirc。

命令行输入命令sudo apt-get install lirc即可。

如果你遇到了连接不上的错误，更换一个安装源就行。

sudo nano /etc/apt/sources.list

录制空调开关命令

这一步骤可以照着网上搜的进行，不过有一些坑要注意。

  1. 生成录制空调命令的模板的时候，可以先拿电视遥控器之类的东西，直接使用空调会因为指令太长失败。

  2. 复制空调指令到模板文件的时候，一定要按照格式来，我前几次没按照格式都是失败的，可以参照一下我的指令文件。（只要照着电视遥控之类的遥控器生成的模板文件填写就行）

  1. 生成的文件记得放到/etc/lirc目录下，我开始没放到这个目录下，一直检测不出来生成的按钮。
